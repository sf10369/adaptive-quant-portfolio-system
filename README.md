# Adaptive Quant Portfolio System

## Overview

This project develops an adaptive portfolio management framework that combines quantitative finance and machine learning techniques to dynamically adjust ETF allocations based on changing market conditions.

The system uses market regime detection and financial feature engineering to identify shifts in market behavior and allocate capital accordingly. Performance is evaluated through historical backtesting and compared against a passive SPY benchmark.

## Key Features

* Machine Learning-Based Market Regime Detection
* Dynamic ETF Allocation
* Portfolio Backtesting Framework
* Risk-Adjusted Performance Evaluation
* Financial Feature Engineering

## Financial Features Engineered

* Momentum Indicators
* Volatility Measures
* Trend Signals
* Drawdown Metrics
* Market State Variables

## Performance Evaluation

The strategy was evaluated using:

* Sharpe Ratio
* Maximum Drawdown
* Cumulative Returns
* Benchmark Comparison (SPY)
